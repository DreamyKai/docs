## Ending a Session

### cURL

```bash
curl -X DELETE -H 'Authorization: Bearer <your-api-key>' \
    https://engine.hyperbeam.com/v0/vm/52f968cb-6739-4197-83d7-2305fe5d6f54
```

### Example response

```json
{
  "session_id": "52f968cb-6739-4197-83d7-2305fe5d6f54"
}
```

| Property     | Type   | Description                               |
| ------------ | ------ | ----------------------------------------- |
| `session_id` | string | The ID of the session that was terminated |
