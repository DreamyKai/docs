## Programmatic control

### cURL

```bash
url -X POST -H 'Authorization: Bearer <your-admin-token>' \
    <embed_url>/tabs.update \
    -d '[1, {"active": true}]'
```

View all the supported [tab methods](https://www.npmjs.com/package/@hyperbeam/web#control-tabs-programmatically) (e.g. tabs.create, tabs.remove).

The request body mirrors the function signature of the [Chrome extension tab API](https://developer.chrome.com/docs/extensions/reference/tabs/#method).
