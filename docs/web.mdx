---
title: Web
---

We provide a [client-side SDK](https://www.npmjs.com/package/@hyperbeam/web) that lets you:

* Control browser navigation programmatically
* Query browser state and listen to events
* Manager user control permissions

The library is published on [npm](https://www.npmjs.com/package/@hyperbeam/web) as @hyperbeam/web

```bash Terminal
npm install @hyperbeam/web --save
```
